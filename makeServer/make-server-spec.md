# Общее описание
Сервер собирается индивидуально под целевой пк на котором установлено бразуерное расширение BTR. Сервер обрабатывает запросы от BTR по средством локального HTTP сервера.

# Управление сервером
    1. Телеграм бот

# Приготовление сервера
Класс ServerPackageBuilder принимает параметры установки, подготавливает и упаковывает сервер под определенную машину.
